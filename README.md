# nlp-semantic-clustering-amazon-food-reviews
Unsupervised topic modeling of Amazon food reviews using SBERT embeddings, K-Means clustering, and KeyBERT for semantic interpretation

The project utilizes the Amazon Fine Food Reviews dataset from Kaggle. Due to the file size (300MB+), the original dataset is not included in this repository.
Dataset: https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews
